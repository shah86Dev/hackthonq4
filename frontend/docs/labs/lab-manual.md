# Lab Manuals

## Module 1: Foundations of Physical AI

### Introduction to Physical AI


#### Setting up ROS 2 Environment

**Duration**: 90 minutes
**Difficulty**: beginner

In this lab, students will install and configure ROS 2 for Physical AI applications

**Instructions:**
- Install ROS 2 Humble Hawksbill
- Set up a workspace
- Create a basic publisher-subscriber package
- Run the example nodes

**Expected Outcomes:**
- ROS 2 environment successfully installed
- Basic publisher-subscriber example running
- Understanding of ROS 2 workspace structure

---
### Sensorimotor Coordination


#### Implementing a Simple Sensorimotor Loop

**Duration**: 120 minutes
**Difficulty**: intermediate

Students will create a basic sensorimotor loop using ROS 2 and Gazebo

**Instructions:**
- Create a ROS 2 package for sensorimotor control
- Implement a basic perception node
- Create an action node
- Connect perception and action in a loop
- Test in Gazebo simulation

**Expected Outcomes:**
- Working sensorimotor loop implementation
- Understanding of feedback control
- Experience with ROS 2 nodes and topics
